# Commodore 128 VDC
for the C128 MiSTer FPGA core, by Erik Scheffers

## Implemented
 * Memory interface (including copy and fill)
 * Standard text mode
 * Attributes, color
 * Cursor
 * 640x200 "standard" graphics mode
 * Lightpen trigger
